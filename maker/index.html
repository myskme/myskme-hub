出题 工坊

网页拼配对 / 选择 / 阅读题 → 导出题库 JSON → 放进书架即时上架 · 免后端

① 题库信息

兑换码 = 卷号（唯一），学生扫码 / 输码即装这套题。

② 配对题 0

词 ↔ 义 记忆题。会自动分成 m1 / m2 / m3 三关。难度：1 基础 · 2 冲刺（可空）。

③ 选择 / 阅读 / 完形题 0

每个「块」= 一段短文（可选）+ 若干小题。纯选择题不填短文即可。会自动分成 m4 / m5 两关（短文不会被拆散）。

校验 · 实时

导出内容（banks/src/<码>.json）

这就是要保存的文件内容，可直接复制。

怎么上架

  1. 点 导出 得到 <兑换码>.json。
  2. 把它放进仓库 banks/src/ 并推送 —— 或用 Pages CMS 的「题库」集合直接填。
  3. GitHub Action 自动构建校验 → 一分钟后 书架 / 词灵对决 / 打印中心 / 每日一题 全部生效。兑换码即卷号。
改现有题库 / 导入 JSON

载入后可修改再导出。注意：分关（m1–m5）会按数量重新平均分配。
