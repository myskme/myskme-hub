# 学院谷地 · 荣誉殿堂（优秀作文展示墙）

单文件离线展示墙：画廊 + 点击放大看王老师点评 + 年级/主题筛选 + 金奖置顶 + 班级口令软门槛 + 密码后台。当前为**样张占位**，等替换为真实扫描件。

线上（占位演示）：`https://myskme.github.io/myskme-hub/wall/`

## 怎么加一篇优秀作文

**方式 A · 管理员模式（最省事，推荐）**
1. 点左下角 ✎ → 密码 `mrwolf4358` → 进入管理员。
2. 「＋ 加作品」→ 选这张作文的**扫描件/照片**（自动缩到 ~1000px 嵌入，不臃肿）。
3. 依次填：年级、题目、作者。
4. 卡片上「改」补：王老师点评、高分亮点、评级、主题；「★ 金奖」可置顶。
5. 「导出 index.html」→ 用导出的文件替换本目录的 `index.html`，重新部署即更新。

**方式 B · 手动清单（适合批量）**
把扫描件放进 `wall/img/`，在 `wall/index.json`（见 `index.sample.json` 模板）加一条：
```json
{ "id":"w10", "title":"My Summer Plan", "author":"初二 · L 同学", "grade":"初二",
  "theme":"假期生活", "date":"2026-07", "score":"A+", "note":"王老师点评……",
  "highlights":["高级句型","结构清晰"], "img":"img/w10.jpg", "featured":true }
```
页面启动会优先读 `wall/index.json`（存在即用，否则用文件内嵌的样张数据）。

## 隐私（涉及未成年人手写件，务必先想清楚）
- 作者**默认用化名 / 学号 / 首字母**，不默认实名（每篇可单独设）。
- 要公开**实名 + 手写真迹**前，建议先征得家长同意。
- 想半公开：管理员「设班级口令」→ 只有拿到口令的人（你班 / 家长群）能进。留空即公开。

## 字段
`title 题目 · author 作者(化名) · grade 年级 · theme 主题 · date 年月 · score 评级(如 A+) · note 王老师点评 · highlights[] 高分亮点 · featured 金奖置顶 · img 扫描件(路径或自动嵌入的 dataURI)`
